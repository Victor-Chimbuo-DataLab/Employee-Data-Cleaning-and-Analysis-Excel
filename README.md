# Employee Data Cleaning and Analysis Project (Excel)

## Project Overview
This project focuses on cleaning, transforming, and analyzing a raw employee dataset using Microsoft Excel.

The objective was to prepare inconsistent data for analysis and apply aggregation techniques to generate meaningful summaries.

---

## Dataset Description
The dataset contains employee records with inconsistencies in text formatting, spacing issues, and structural irregularities.

---

## Data Cleaning & Transformation

The dataset was cleaned and standardized using the following techniques:

### Text Functions
- TRIM (to remove extra spaces)
- UPPER, LOWER, PROPER (for text standardization)
- CONCAT (to combine fields)
- LEN (for validation and consistency checks)

### Data Structuring
- Used **Text to Columns** to split and organize data fields

---

## Data Analysis

Performed aggregations and calculations using:

- SUM
- SUMIFS
- AVERAGE
- AVERAGEIFS
- MAXIFS
- COUNTA

These were used to summarize employee data and extract key metrics.

---

## Key Outcome
- Transformed raw, inconsistent data into a structured format
- Applied aggregation functions to generate meaningful summaries
- Prepared dataset for reporting and further analysis

- Aggregation & Conditional Functions
